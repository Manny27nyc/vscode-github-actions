# GitHub Actions

Syntax Highlight and Snippet for GitHub Actions `main.workflow` file

## Usage

* Syntax Highlight

  Open `main.workflow` file to see syntax highlight

* Snippet

  Type `workflow` and `action` to trigger code snippet.
